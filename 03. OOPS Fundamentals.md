# 2ï¸âƒ£ Procedure-Oriented Programming (POP) vs Object-Oriented Programming (OOP)

## POP â€“ Simple Meaning
POP focuses on **functions and steps**.

## Deep Explanation
- Data is usually global
- Any function can modify it
- As system grows, debugging becomes hard

### C# Example (POP)
```csharp

```

---

## OOP â€“ Simple Meaning
OOP focuses on **objects**, each responsible for its own data and behavior.

## Deep Explanation
- Data is protected inside objects
- Responsibilities are clearly separated
- Systems become scalable and maintainable

### C# Example (OOP)
```csharp
class Account
{
    private int balance = 1000;
    public void Deposit(int amt) { balance += amt; }
}
```
